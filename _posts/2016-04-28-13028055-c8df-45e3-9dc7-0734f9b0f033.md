# Brocade Cheet Sheet

\# Active zone set  
\> **cfgactshow | grep cfg**

\# Show all zone set and active  
\> **zoneshow | grep 'cfg\\|config'**

\# delete ZoneSet  
\> **cfgDelete <zoneSet\>**

\# delete Zone  
\> **zoneDelete <zoneName\>**

\# Remove Zone member from ZoneSet  
\> **cfgRemove <zoneSet\>,ZoneName1;ZoneName2**

\# Save Configuration to non-volatile memory  
\> **cfgSave**

\# Enable Config  
\> **cfgEnable <zoneSet\>**
